# Tingala Tech Website

## Overview
Tingala Tech is a no-code IT service provider based in Puerto Princesa, focusing on elevating business and device performance through reliable and affordable services offered by local students. This website serves as a platform to showcase our services, pricing, and contact information.

## Project Structure
The project is organized as follows:

```
tingala-tech-website
├── src
│   ├── index.html               # Main HTML document
│   ├── styles                   # Stylesheets
│   │   ├── main.css             # Primary styles
│   │   └── utilities.css        # Utility classes
│   ├── scripts                  # JavaScript files
│   │   └── main.js              # Interactivity scripts
│   ├── data                     # Data files
│   │   └── services.json        # Service packages and pricing
│   ├── components               # HTML components
│   │   ├── hero.html            # Hero section
│   │   ├── services.html        # Services and pricing section
│   │   ├── why-choose.html      # Why choose section
│   │   └── contact.html         # Contact section
│   └── forms                    # Forms
│       └── client-intake-google-form.md # Client intake form
├── package.json                 # npm configuration
├── .gitignore                   # Version control ignores
└── README.md                    # Project documentation
```

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd tingala-tech-website
   ```

2. **Install Dependencies**
   Ensure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Run the Project**
   Open `src/index.html` in your web browser to view the website.

## Features
- **Hero Section**: Engaging introduction with a clear call-to-action.
- **Services & Pricing**: Detailed list of services with transparent pricing.
- **Why Choose Us**: Highlights the reliability and local focus of Tingala Tech.
- **Contact Form**: Easy way for clients to reach out for services.
